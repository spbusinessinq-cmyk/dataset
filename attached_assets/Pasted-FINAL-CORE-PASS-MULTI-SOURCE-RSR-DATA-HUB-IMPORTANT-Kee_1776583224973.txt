4. UNIFIED SIGNAL OBJECT
==================================================

All data entering the system should normalize into one common object shape, for example:

{
  "id": "SG-0049",
  "title": "...",
  "summary": "...",
  "rawText": "...",
  "source": "Reuters",
  "sourceType": "News",
  "classification": "WATCH",
  "confidence": 72,
  "engine": "INGEST",
  "tags": [],
  "entities": [],
  "systemImpact": [],
  "timestamp": "...",
  "status": "pulled"
}

This is critical.
No matter what enters the hub, it should become a signal candidate.
